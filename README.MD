# Телеграм бот.
## Для быстрого перехода.
*  Важно 
    Для Tелеграм Бота нужно в корне преокта создать файл .env и указать токен как в примерном файле .env.template
## Описание
    1. На любое сообщение приходит ответ ID пользователя.
    2. Последнее посещение хранится в базе данных.
1. Полезные команды
   + /start - Приветствие("Hello,{user}")
   + /Button - Активные кнопки для перехода по разделам
2. Реализованы разделы:
    - Главное меню
    - Развлекательный
    - Учебный
3. Docker compose:
    - pg
    - pgadmin
